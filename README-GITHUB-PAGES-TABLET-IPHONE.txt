APP BRAGA - GITHUB PAGES (TABLET + IPHONE)

Esta versão já está preparada para usar no:
- Tablet Android
- iPhone
- Browser web
- GitHub Pages

COMO PUBLICAR NO GITHUB:
1. Cria um repositório novo no GitHub
2. Faz upload de TODOS os ficheiros desta pasta para a raiz do repositório
3. Vai a:
   Settings > Pages
4. Em "Build and deployment":
   Source = Deploy from a branch
5. Branch = main
6. Folder = /root
7. Guarda

O teu link vai ficar tipo:
https://TEU-UTILIZADOR.github.io/NOME-DO-REPOSITORIO/

NO IPHONE:
1. Abre o link no Safari
2. Clica em Partilhar
3. "Adicionar ao Ecrã principal"

NO TABLET:
1. Abre o link no Chrome
2. Usa normalmente como app web

NOTAS:
- OCR e câmara funcionam melhor por HTTPS
- GitHub Pages resolve isso automaticamente
- Esta versão inclui:
  * scanner por código
  * OCR por foto da folha
  * data da folha
  * data do scan
  * localização automática por série
